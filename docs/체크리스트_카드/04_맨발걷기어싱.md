# 04. 맨발 걷기 / 어싱 (Barefoot Walking · Earthing)
**유형:** 경도 유산소 · 이완

**기전.** 경도 유산소 + 접지·감각 자극 + 광노출의 활동·이완 복합 세션.

**핵심 변수(●):** 걸음수/이동거리 · 활동 MET · HR(실시간) · SpO2 · 좌식시간(↓)
**보조 변수(○):** 세션 후 HRV · RHR · 호흡수

**기대 반응 →** 걸음수 ▲ · 활동 MET ▲ · 걷기 중 HR 적정대 · 세션 후 HRV ▲(반등)

## 측정 체크리스트 (T0~T4)
- [ ] T0 baseline 캡처
- [ ] T1 걷기 중 걸음수·이동·HR·MET 연속 기록
- [ ] T1 SpO2 측정 (호흡·순환)
- [ ] T2 종료 직후 1분 회복 심박 캡처
- [ ] T3 +30분 HRV 반등·RHR 캡처
- [ ] 당일 좌식시간 감소량 기록

**주관적 자기보고:** 상쾌함 · 발/신체 감각 · 기분 VAS
**양호 판정 기준:** 목표 걸음수 달성 + 1분 회복 심박 −12bpm↑ 또는 세션 후 HRV ▲

*측정 시점 — T0 세션 직전 / T1 세션 중 / T2 종료 직후 / T3 +30분 / T4 익일 아침*
