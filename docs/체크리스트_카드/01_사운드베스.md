# 01. 사운드베스 (Sound Bath)
**유형:** 고이완 · 저활동

**기전.** 저주파 진동·배음이 부교감 신경을 활성화하여 깊은 이완을 유도한다.

**핵심 변수(●):** HRV · RHR · 호흡수 · 스트레스 점수 · HR(실시간)
**보조 변수(○):** SpO2 · 피부온도 · 당일 수면 잠복기

**기대 반응 →** HRV ▲ · RHR ▼ · 호흡수 ▼ · HR ▼ · 스트레스 ▼ · 입면시간 ▼

## 측정 체크리스트 (T0~T4)
- [ ] T0 baseline (누운 자세, 안정 5분) 캡처
- [ ] T1 세션 중 HR·호흡 연속 기록 (하강 곡선 확인)
- [ ] T2 종료 직후 HRV·RHR·호흡수 캡처
- [ ] T3 +30분 이완 지속 여부 캡처
- [ ] T4 당일 밤 수면 잠복기·수면 효율 확인

**주관적 자기보고:** 이완도 · 긴장이완 VAS (前/後)
**양호 판정 기준:** HRV +10%↑ 또는 호흡수 −2brpm↓ 또는 스트레스 −10점↓

*측정 시점 — T0 세션 직전 / T1 세션 중 / T2 종료 직후 / T3 +30분 / T4 익일 아침*
