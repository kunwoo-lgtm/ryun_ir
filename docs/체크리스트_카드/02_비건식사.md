# 02. 비건 식사 (Vegan Meal)
**유형:** 대사 · 회복

**기전.** 식이 구성 변화가 식후 대사·염증·수면 질에 영향. 링은 식후 심박 반응·수면·체온 중심 측정(CGM 옵션 병행).

**핵심 변수(●):** 피부온도 편차 · 야간 HRV · 수면 효율/구조 · 식후 HR 반응
**보조 변수(○):** RHR · 스트레스 점수 · (옵션) CGM 혈당곡선

**기대 반응 →** 식후 HR 급등 완화 · 야간 HRV ▲ · 수면 효율 ▲ · 피부온도 안정

## 측정 체크리스트 (T0~T4)
- [ ] 식전 T0 HR·baseline 캡처
- [ ] 식후 T2·T3(+30~60분) HR·움직임 캡처 (식후 부하 곡선)
- [ ] (옵션) CGM 혈당 피크·회복 시간 기록
- [ ] T4 익일 아침 야간 HRV·수면 효율·구조 확인
- [ ] 피부온도 편차(수면 중) 기록

**주관적 자기보고:** 포만감 · 소화 편안함 · 식후 각성도 VAS
**양호 판정 기준:** 야간 HRV 대비 ▲ 또는 수면 효율 +3%p↑, 식후 HR 피크 완만

*측정 시점 — T0 세션 직전 / T1 세션 중 / T2 종료 직후 / T3 +30분 / T4 익일 아침*
