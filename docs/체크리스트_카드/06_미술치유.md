# 06. 미술 치유 (Art Therapy)
**유형:** 정서 조절 · 몰입

**기전.** 창작 몰입(flow)을 통한 정서 조절·표현. 저활동·정서 중심 세션.

**핵심 변수(●):** HRV · 스트레스 점수 · HR 안정도(변동폭)
**보조 변수(○):** 호흡수 · 좌식시간 · 준비도 점수

**기대 반응 →** 몰입 중 HR 변동폭 ▼(안정) · HRV ▲ · 스트레스 ▼

## 측정 체크리스트 (T0~T4)
- [ ] T0 baseline 캡처
- [ ] T1 창작 중 HR 연속 기록 (몰입 시 변동성 안정 확인)
- [ ] T2 종료 직후 HRV·스트레스 점수 캡처
- [ ] T3 +30분 정서 안정 지속 캡처
- [ ] 작품 기반 감정 라벨링 기록(인지적 자비)

**주관적 자기보고:** 몰입도(flow) · 감정 표현 · 정서 해소 VAS (前/後)
**양호 판정 기준:** 세션 중 HR 변동폭 축소 + HRV ▲ 또는 스트레스 −8점↓

*측정 시점 — T0 세션 직전 / T1 세션 중 / T2 종료 직후 / T3 +30분 / T4 익일 아침*
