# 07. 댄스 만달라 (Dance Mandala)
**유형:** 중강도 활동 · 정서 방출

**기전.** 유산소 움직임 + 리듬 + 정서 방출(카타르시스)로 각성 후 부교감 반등.

**핵심 변수(●):** HR(피크·평균) · 활동 MET/칼로리 · 세션 후 HRV(반등) · 회복 점수 · 좌식(↓)
**보조 변수(○):** 야간 HRV · RHR · 호흡수 · 스트레스 점수

**기대 반응 →** 세션 중 HR ▲(유산소) → 세션 후 빠른 회복 · HRV 반등 ▲ · 정서 해소

## 측정 체크리스트 (T0~T4)
- [ ] T0 baseline 캡처
- [ ] T1 댄스 중 HR(피크/평균)·MET·칼로리·움직임 연속 기록
- [ ] T2 종료 직후 HR·1분 회복 심박 캡처
- [ ] T3 +30분 HRV 반등·부교감 회복 캡처
- [ ] T4 익일 야간 HRV·회복 점수 확인

**주관적 자기보고:** 활력 · 정서 방출 · 기분 상승 VAS (前/後)
**양호 판정 기준:** 유효 유산소대 도달 + 1분 회복 심박 −15bpm↑ 또는 T3 HRV 반등 ▲

*측정 시점 — T0 세션 직전 / T1 세션 중 / T2 종료 직후 / T3 +30분 / T4 익일 아침*
