# 05. 텃밭 가꾸기 (Gardening)
**유형:** 저강도 활동 · 정서 안정

**기전.** 저강도 지속 활동 + 흙 접촉 + 햇빛 + 성취감.

**핵심 변수(●):** 활동 MET/시간 · 걸음수 · 좌식시간(↓) · 스트레스 점수
**보조 변수(○):** HR(실시간) · 피부온도(광노출) · SpO2 · 수면 효율

**기대 반응 →** 활동시간 ▲ · 좌식 ▼ · 스트레스 ▼ · 저강도 안정 HR 유지

## 측정 체크리스트 (T0~T4)
- [ ] T0 baseline 캡처
- [ ] T1 작업 중 활동시간·MET·걸음수·HR 연속 기록
- [ ] T1 햇빛 노출 시간(광노출) 메모
- [ ] T2 종료 직후 스트레스 점수·HR 캡처
- [ ] 당일 좌식시간 감소·총 활동시간 기록
- [ ] T4 당일 밤 수면 효율 확인

**주관적 자기보고:** 성취감 · 정서 안정 · 몰입 VAS
**양호 판정 기준:** 활동시간 목표 달성 + 스트레스 −10점↓ 또는 좌식 −60분↓

*측정 시점 — T0 세션 직전 / T1 세션 중 / T2 종료 직후 / T3 +30분 / T4 익일 아침*
