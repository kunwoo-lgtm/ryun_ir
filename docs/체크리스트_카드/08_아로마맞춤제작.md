# 08. 아로마 맞춤 제작 (Custom Aroma Blending)
**유형:** 후각·정서 조절 · 저활동

**기전.** 후각 자극이 변연계(편도체·해마)에 직접 작용해 자율신경·정서를 즉각 조절한다. 향을 직접 고르고 배합하는 과정은 선택 주도성·창작 몰입을 동반한다.

**핵심 변수(●):** HRV · 호흡수 · 스트레스 점수 · HR(향 흡입 반응)
**보조 변수(○):** RHR · 피부온도 · 준비도/회복 점수 · 수면 잠복기

**기대 반응 →** 이완 블렌드 → HRV ▲·호흡 ▼·스트레스 ▼  /  활력 블렌드 → 흡입 시 HR ▲(적정 각성)·각성도 ▲

## 측정 체크리스트 (T0~T4)
- [ ] T0 baseline (좌위, 안정 5분) 캡처
- [ ] T1 오일별 시향 시 실시간 HR·호흡 반응을 오일마다 캡처(비교표)
- [ ] T1 블렌딩 몰입 중 HR 변동폭 안정 확인
- [ ] T2 최종 맞춤 블렌드 흡입 직후 HRV·호흡·스트레스 캡처
- [ ] T3 +30분 이완/각성 지속 캡처
- [ ] 취침 전 디퓨징 시 수면 잠복기·입면 확인(이완 블렌드)
- [ ] 개인 맞춤 블렌드 레시피 기록(오일·비율·목적) → BioPassport 저장
