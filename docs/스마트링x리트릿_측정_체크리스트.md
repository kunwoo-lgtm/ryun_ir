# 스마트링 × 리트릿 프로그램 통합 측정 체크리스트
### — 하이테크 바이오 데이터 × 7개 리트릿 프로그램의 조합별 프로토콜 —

> **목적.** 스마트링(예: Oura·Ultrahuman·Galaxy Ring 등)으로 측정 가능한 신체 데이터를 표준화하고, 리트릿 7개 프로그램 각각에 대해 "무엇을·언제·어떤 방향으로" 측정할지를 체크리스트로 규정한다. 이는 RYUN 리트릿의 `BioPassport` 회복 궤적 데이터의 수집 규격이자, 임팩트 지표 공시의 원천 데이터가 된다.

---

## 0. 사용법 개요

- 프로그램마다 **측정 시점(T0~T4)**에 스마트링 데이터를 캡처하고 체크박스로 기록한다.
- **핵심 변수(●)**는 필수 측정, **보조 변수(○)**는 여건 시 측정, **정성(자기보고)**는 매 세션 필수.
- 판정은 "세션 전(T0) 대비 변화(Δ)"로 하며, 개인 baseline 대비 상대값을 우선한다.
- 스마트링만으로 측정 불가한 지표(코르티솔·IL-6·혈당 등)는 §1.2의 옵션 모듈로 별도 수집한다.

### 측정 시점 정의 (모든 프로그램 공통)

| 코드 | 시점 | 방법 |
|---|---|---|
| **T0** | 세션 시작 직전 (안정 5분 후) | 착석/누운 상태 baseline 측정 |
| **T1** | 세션 진행 중 | 실시간 HR·움직임 연속 기록 |
| **T2** | 세션 종료 직후 | 즉시 스냅샷 |
| **T3** | 종료 +30분 | 회복 반등(rebound) 측정 |
| **T4** | 익일 아침 기상 시 | 야간 수면·회복 반영값 |

## 1. 스마트링 측정 변수 사전

### 1.1 측정 가능 변수 (5개 군)

**A. 심혈관·자율신경 (Cardiovascular / Autonomic)**

| 변수 | 단위 | 의미 | 스트레스↓ 방향 |
|---|---|---|---|
| HRV (RMSSD) | ms | 심박변이도 — 부교감/회복 핵심 지표 | ▲ 상승 |
| RHR (안정시 심박) | bpm | 자율신경 부하·회복도 | ▼ 하강 |
| HR (실시간 심박) | bpm | 순간 각성/이완 | 상황별 |
| 호흡수 | brpm | 이완·부교감 활성 | ▼ 하강 |

**B. 산소·체온·대사 (Oxygen / Temp / Metabolic)**

| 변수 | 단위 | 의미 |
|---|---|---|
| SpO2 (산소포화도) | % | 호흡·순환 효율 (95%↑ 정상) |
| 피부 온도 편차 | °C | baseline 대비 편차 — 회복·염증·생리주기 |
| 소모 칼로리 / MET | kcal / MET | 대사·활동 강도 |

**C. 수면 (Sleep, T4 반영)**

| 변수 | 단위 | 의미 |
|---|---|---|
| 총 수면시간 / 효율 | h / % | 회복 총량 |
| 수면 구조 (深/REM/얕은) | % | 회복 질 |
| 수면 잠복기 | min | 입면 용이성 (이완 효과) |
| 야간 HRV / 야간 RHR | ms / bpm | 밤 사이 회복력 |

**D. 활동·움직임 (Activity)**

| 변수 | 단위 | 의미 |
|---|---|---|
| 걸음수 / 이동거리 | 보 / km | 활동량 |
| 활동 강도 (MET-분) | MET·min | 유산소 부하 |
| 좌식 시간 | min | 비활동 부담 (↓ 목표) |
| 움직임 지수 (가속도) | a.u. | 동작 리듬·강도 |

**E. 종합 스코어 (Derived)**

| 변수 | 범위 | 의미 |
|---|---|---|
| 준비도 / 회복 점수 | 0–100 | 종합 회복 상태 (▲ 목표) |
| 스트레스 점수 | 0–100 | 종합 부하 (▼ 목표) |
| 수면 점수 | 0–100 | 종합 수면 질 |

### 1.2 옵션 모듈 (스마트링 측정 불가 — 별도 수집)

- ☐ **타액 코르티솔** (스트레스 호르몬, 세션 前/後) — CBCT 근거지표(Pace et al.)와 직접 대응
- ☐ **혈중/건조혈반 IL-6** (염증) — 심화 리서치용, 주기적 샘플
- ☐ **연속혈당(CGM)** — 비건 식사 프로그램 대사 반응 측정 시
- ☐ **혈압** (커프) — 사운드베스·명상 이완 검증 보조
- ☐ **표준 설문**: 스트레스(PSS)·외로움(UCLA)·자비심(CLHS)·기분(POMS) — 입·퇴소 시

## 2. 공통 측정 프로토콜 (모든 세션 반복)

- [ ] 스마트링 착용·동기화 상태 확인 (배터리 30%↑, 밀착 확인)
- [ ] 참가자 ID / 프로그램명 / 날짜·시각 기록
- [ ] **T0** baseline 측정 (안정 5분 후 HRV·RHR·호흡·HR 캡처)
- [ ] **T1** 세션 중 실시간 HR·움직임 연속 기록 시작
- [ ] **T2** 종료 직후 스냅샷 캡처
- [ ] **T3** +30분 회복값 캡처
- [ ] **T4** 익일 아침 수면·야간 회복값 확인·기록
- [ ] 정성 자기보고(이완도·기분 VAS 0–10) 세션 前/後 기록
- [ ] 특이사항(카페인·수면부족·생리주기·질환) 메모
- [ ] 데이터 시트(§6) 입력 및 BioPassport 업로드

## 3. 프로그램별 측정 체크리스트

### 3.1 사운드베스 (Sound Bath)

**기전.** 저주파 진동·배음이 부교감 신경을 활성화 → 깊은 이완. **저활동·고이완** 세션.

**핵심 변수(●):** HRV ● · RHR ● · 호흡수 ● · 스트레스 점수 ● · HR(실시간) ●
**보조 변수(○):** SpO2 ○ · 피부온도 ○ · 당일 수면 잠복기 ○

**기대 반응:** HRV ▲ · RHR ▼ · 호흡수 ▼ · HR ▼ · 스트레스 ▼ · 입면 시간 ▼

- [ ] T0 baseline (누운 자세, 안정 5분) 캡처
- [ ] T1 세션 중 HR·호흡 연속 기록 (하강 곡선 확인)
- [ ] T2 종료 직후 HRV·RHR·호흡수 캡처
- [ ] T3 +30분 이완 지속 여부 캡처
- [ ] T4 당일 밤 수면 잠복기·수면 효율 확인
- [ ] 자기보고: 이완도·긴장이완 VAS (前/後)
- [ ] **양호 판정:** HRV +10%↑ 또는 호흡수 −2brpm↓ 또는 스트레스 −10점↓

---

### 3.2 비건 식사 (Vegan Meal)

**기전.** 식이 구성 변화 → 식후 대사·염증·수면 질 영향. 스마트링은 **식후 심박 반응·수면·체온** 중심으로 측정(직접 영양 측정 불가 → CGM 옵션 병행).

**핵심 변수(●):** 피부온도 편차 ● · 야간 HRV ● · 수면 효율/구조 ● · 식후 HR 반응 ●
**보조 변수(○):** RHR ○ · 스트레스 점수 ○ · (옵션) CGM 혈당곡선 ○

**기대 반응:** 식후 HR 급등 완화 · 야간 HRV ▲ · 수면 효율 ▲ · 피부온도 안정

- [ ] 식전 T0 HR·baseline 캡처
- [ ] 식후 T2(직후)·T3(+30~60분) HR·움직임 캡처 (식후 부하 곡선)
- [ ] (옵션) CGM 혈당 피크·회복 시간 기록
- [ ] T4 익일 아침 야간 HRV·수면 효율·수면 구조 확인
- [ ] 피부온도 편차(수면 중) 기록
- [ ] 자기보고: 포만감·소화 편안함·식후 각성도 VAS
- [ ] **양호 판정:** 야간 HRV baseline 대비 ▲ 또는 수면 효율 +3%p↑, 식후 HR 피크 완만

---

### 3.3 요가 · 명상 (Yoga & Meditation)

**기전.** 호흡 조절·이완 동작 → 자율신경 균형. **명상(고이완) + 동작 요가(경도 활동)** 이중 성격.

**핵심 변수(●):** HRV ● · 호흡수 ● · 야간 HRV ● · 준비도/회복 점수 ● · 스트레스 점수 ●
**보조 변수(○):** HR(실시간) ○ · 활동 MET ○ · SpO2 ○

**기대 반응:** HRV ▲ · 호흡수 ▼ · 스트레스 ▼ · 회복 점수 ▲

- [ ] T0 baseline (좌위, 안정 5분) 캡처
- [ ] T1 명상 구간 HR·호흡 연속 기록 (안정화 확인)
- [ ] T1 동작 요가 구간 활동 MET·HR 기록
- [ ] T2 종료 직후 HRV·호흡수 캡처
- [ ] T3 +30분 회복 지속 캡처
- [ ] T4 익일 회복/준비도 점수·야간 HRV 확인
- [ ] 자기보고: 집중도·평온함 VAS (前/後)
- [ ] **양호 판정:** HRV +15%↑ 또는 호흡수 −3brpm↓ 또는 회복 점수 +5점↑

---

### 3.4 맨발 걷기 / 어싱 (Barefoot Walking · Earthing)

**기전.** 경도 유산소 + 접지·감각 자극 + 광노출. **활동 + 이완** 복합.

**핵심 변수(●):** 걸음수/이동거리 ● · 활동 MET ● · HR(실시간) ● · SpO2 ● · 좌식시간(↓) ●
**보조 변수(○):** 세션 후 HRV ○ · RHR ○ · 호흡수 ○

**기대 반응:** 걸음수 ▲ · 활동 MET ▲ · 걷기 중 HR 적정대(유산소) · 세션 후 HRV ▲(이완 반등)

- [ ] T0 baseline 캡처
- [ ] T1 걷기 중 걸음수·이동거리·HR·MET 연속 기록
- [ ] T1 SpO2 측정 (호흡·순환)
- [ ] T2 종료 직후 HR 회복 속도(1분 회복 심박) 캡처
- [ ] T3 +30분 HRV 반등·RHR 캡처
- [ ] 당일 좌식시간 감소량 기록
- [ ] 자기보고: 상쾌함·발/신체 감각·기분 VAS
- [ ] **양호 판정:** 목표 걸음수 달성 + 1분 회복 심박 −12bpm↑ 또는 세션 후 HRV ▲

---

### 3.5 텃밭 가꾸기 (Gardening)

**기전.** 저강도 지속 활동 + 흙 접촉 + 햇빛 + 성취감. **저강도 활동 + 정서 안정**.

**핵심 변수(●):** 활동 MET/시간 ● · 걸음수 ● · 좌식시간(↓) ● · 스트레스 점수 ●
**보조 변수(○):** HR(실시간) ○ · 피부온도(광노출) ○ · SpO2 ○ · 수면 효율 ○

**기대 반응:** 활동시간 ▲ · 좌식 ▼ · 스트레스 ▼ · 저강도 안정 HR 유지

- [ ] T0 baseline 캡처
- [ ] T1 작업 중 활동시간·MET·걸음수·HR 연속 기록
- [ ] T1 햇빛 노출 시간(광노출) 메모
- [ ] T2 종료 직후 스트레스 점수·HR 캡처
- [ ] 당일 좌식시간 감소·총 활동시간 기록
- [ ] T4 당일 밤 수면 효율 확인 (신체활동 효과)
- [ ] 자기보고: 성취감·정서 안정·몰입 VAS
- [ ] **양호 판정:** 활동시간 목표 달성 + 스트레스 −10점↓ 또는 좌식 −60분↓

---

### 3.6 미술 치유 (Art Therapy)

**기전.** 창작 몰입(flow) → 정서 조절·표현. **저활동·정서 중심** 세션.

**핵심 변수(●):** HRV ● · 스트레스 점수 ● · HR 안정도(변동폭) ●
**보조 변수(○):** 호흡수 ○ · 좌식시간 ○ · 준비도 점수 ○

**기대 반응:** 몰입 중 HR 변동폭 ▼(안정) · HRV ▲ · 스트레스 ▼

- [ ] T0 baseline 캡처
- [ ] T1 창작 중 HR 연속 기록 (몰입 시 HR 변동성 안정 확인)
- [ ] T2 종료 직후 HRV·스트레스 점수 캡처
- [ ] T3 +30분 정서 안정 지속 캡처
- [ ] 자기보고: 몰입도(flow)·감정 표현·정서 해소 VAS (前/後)
- [ ] 정성: 작품 기반 감정 라벨링 기록 (인지적 자비 측면)
- [ ] **양호 판정:** 세션 중 HR 변동폭 축소 + HRV ▲ 또는 스트레스 −8점↓

---

### 3.7 댄스 만달라 (Dance Mandala)

**기전.** 유산소 움직임 + 리듬 + 정서 방출(카타르시스) → 각성 후 부교감 반등. **중강도 활동 + 정서 방출**.

**핵심 변수(●):** HR(피크·평균) ● · 활동 MET/칼로리 ● · 세션 후 HRV(반등) ● · 회복 점수 ● · 좌식(↓) ●
**보조 변수(○):** 야간 HRV ○ · RHR ○ · 호흡수 ○ · 스트레스 점수 ○

**기대 반응:** 세션 중 HR ▲(유산소) → 세션 후 HR 빠른 회복 · HRV 반등 ▲ · 정서 해소

- [ ] T0 baseline 캡처
- [ ] T1 댄스 중 HR(피크/평균)·MET·칼로리·움직임 연속 기록
- [ ] T2 종료 직후 HR·1분 회복 심박 캡처
- [ ] T3 +30분 HRV 반등·부교감 회복 캡처
- [ ] T4 익일 야간 HRV·회복 점수 확인
- [ ] 자기보고: 활력·정서 방출·기분 상승 VAS (前/後)
- [ ] **양호 판정:** 유효 유산소대 도달 + 1분 회복 심박 −15bpm↑ 또는 T3 HRV 반등 ▲

## 7. 판정·해석 참고 기준 (일반 가이드)

> 아래 임계치는 개인차가 크므로 **개인 baseline 대비 상대 변화**를 우선한다. 반복 측정으로 개인 기준선을 확보한 뒤 적용할 것.

| 지표 | "이완/회복 양호" 참고 기준 (세션 전후) |
|---|---|
| HRV (RMSSD) | +10% 이상 상승 |
| RHR | −3 bpm 이상 하강 |
| 호흡수 | −2 brpm 이상 하강 |
| 1분 회복 심박 (운동 후) | −12 bpm 이상 하강 |
| 스트레스 점수 | −10점 이상 하강 |
| 회복/준비도 점수 | +5점 이상 상승 |
| 수면 효율 | +3%p 이상 상승 |

**주의.** 카페인·음주·수면부족·생리주기·발열·질환은 지표를 교란하므로 반드시 특이사항에 기록하고 해석에 반영한다. 스마트링 값은 의료 진단이 아닌 **경향 관찰용**이다.
